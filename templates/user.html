常用数据形式

Hell {{ name }}!

字典:{{ mydict["key"] }}

列表:{{ mylist }}

元组:{{ mytuple }}

数字:{{ myint }}

对象方法的返回值:{{ u.my_method() }}

变量过滤器

safe:渲染时不对标签进行转义{{ h1|safe }}

capitalize:把值的首字母转换为大写，其他字母转换为小写{{ words|capitalize }}

lower:把值转换为小写形式{{ words|lower }}

upper:把值转换为大写{{ words|upper }}

title：把值中的每个单词的首字母转换为大写{{ words|title }}

trim：把值的首尾空格去掉{{ words|trim }}

striptags：渲染之前把值中的所有HTML标签全删掉{{ h1|striptags }}

选择结构

{% if admin %} hello,{{ name }} {% elif age %} 今年：{{ age }} {% else %} hello,Stranger! {% endif %}

循环结构

    {% for foo in fruit %}
  • {{ foo }}
    • {% endfor %}

宏扩展

{% macro render_comment(comment)%}
  • {{ comment }}
    • {% endmacro %}
      {% for f in fruit %}
    1. {{ render_comment(f) }}
      2. {% endfor %}
